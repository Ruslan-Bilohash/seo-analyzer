О проекте

SEO Analyzer — это современный, быстрый и полностью бесплатный онлайн-инструмент для анализа основных SEO-параметров любой веб-страницы.

Он помогает веб-мастерам, SEO-специалистам и владельцам сайтов быстро находить проблемы и получать конкретные рекомендации по улучшению.

Возможности

Title & Meta

Проверка длины Title, Description, Keywords, Robots, Open Graph

H1 / H2

Количество заголовков и рекомендации

Изображения

Проверка alt-тегов у всех картинок

Скорость

Время загрузки + размер контента

Ссылки & URL

Внутренние / внешние / nofollow + читаемость URL

Как установить

  1. Скачай все файлы проекта
  2. Загрузи их на хостинг с поддержкой PHP 7.4+
  3. Убедись, что файлы seo_history.json и subscribe.json доступны для записи
  4. Открой в браузере index.php
